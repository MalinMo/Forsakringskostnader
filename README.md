# Forsakringskostnader
Individuell inlämningsuppgift

## Körinstruktion
**Öppna projektet**
Öppna filen försäkringsanalys.Rmd i RStudio.

**Paket som används**
Installera paketen om du inte redan har dem:
```r
install.packages("tidyverse")
install.packages("skimr")
install:pacakges("patchwork")
install.packages("broom")
```

**Kör analysen**
Klicka på Knit i filen forsakringsanalys.Rmd i RStudio för att rendera dokumentet. En HTML-fil skapas automatiskt i samma mapp.
